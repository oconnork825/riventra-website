# Riventra Website v2

Modern Next.js site for Riventra LLC.

## Live site
https://riventra.app

## Local development
```bash
npm install
npm run dev
```

## Deploy
Import this repo into Vercel. Vercel will automatically build and deploy.

## Structure
- `app/page.jsx` — main website
- `app/layout.jsx` — metadata/layout
- `app/globals.css` — global styling
